Drop your .glb 3D models here.

Expected by the current projects.js:
  robotic-arm.glb   → shown in the 7-DOF Robotic Arm detail window's 3D viewer

How to make a .glb from your CAD:
  - Fusion 360:  File → Export → choose "GLTF Binary (*.glb)"
  - Blender:     import STEP (built-in in 4.x / add-on) → File → Export → glTF 2.0 (.glb)
  - Keep it lightweight: decimate/merge tiny fasteners, aim for < ~15 MB for fast loading.

Until a .glb exists here, <model-viewer> simply shows the project's poster image,
so the site works fine without it.
